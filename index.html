新兵大力的博客初体验

Posted on 2019-07-29

第一步：安装hexo框架

①下载node.js长期版本

②管理员模式打开cmd后输入node -v查看是否下载成功

③利用npm来安装cnpm，此处利用淘宝镜像速度更快
命令：npm install -g cnpm –registry=https://registry.npm.taobao.org
安装完成后，记得将cnpm安装路径添加到系统环境变量，可用cnpm -v验证是否安装成功

④安装hexo
命令：cnpm install -g hexo-cli

⑤验证
命令：hexo -v

第二步：写下第一篇博客并部署到本地

①切换到自己的用户路径，可自选
命令：cd C:\Users\排骨君

②创建一个专用博客文件夹，并在其中初始化一个hexo博客框架
命令：mkdir blog
cd blog
hexo init

③启动本地服务器，用于预览主题。默认地址： http://localhost:4000/
命令：hexo s

④新建一篇文章
命令：hexo new “我的博客初体验”
此命令会显示创建路径，win下可用文档编辑器打开md文件并编辑

④编辑完成后清除缓存文件 (db.json) 和已生成的静态文件 (public)（一般在博客编辑完成或更换主题后）
命令:hexo clean

⑥生成静态文件
命令：hexo g

⑦启动服务器，可以看到此时已经将文章部署到本地了
命令：hexo s

第三步：将博客发布到上github上

①在blog文件夹下安装git部署插件
命令： cnpm install hexo-deployer-git –save

②修改配置文件_config.yml
找到Deployment部分，设置成如下：
deploy:
type: git
repo: https://github.com/rookiedali/rookiedali.github.io.git
branch: master
注意每个冒号之后都有个空格
地址贴上自己github上博客项目网址

③部署到远端,此时就可以直接访问自己的github网站来看博客了
命令：hexo d

④修改博客主题风格(这里是直接克隆github上大神做个一个主题)
命令：git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia

⑤修改配置文件_config.yml
找到Extensions部分下的theme，将内容中的landscape改为yilia

⑥清理缓存并生成
命令：hexo clean
hexo g

⑥启动服务器，可以看到此时已经将文章部署到本地了
命令：hexo s

⑦部署到远端,此时就可以直接访问自己的github网站来看博客了
命令：hexo d
